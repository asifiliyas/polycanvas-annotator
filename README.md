# ⚾ PolyCanvas Pro

A high-performance baseball image annotation tool built with React and TypeScript. This application meets all core requirements of the frontend assignment and extends them with production-level features.

## ✅ Core Requirements Met
- **Image Display**: Renders a sample baseball image centered on the page.
- **Polygon Annotation**: Interactive vertex placement with automatic line connectors.
- **Closure Logic**: Hover highlighting and click-to-close functionality on the first vertex.
- **Reset**: Full workspace reset capability.
- **Export**: Exports precise polygon data as structured JSON.
- **SVG Engine**: Utilizes SVG for resolution-independent, interactive graphics.



## --------------------------------------------------- [Added Extra Features] ---------------------------------------------------

- **Undo/Redo (Ctrl+Z / Ctrl+Y)**: Robust 50-step action history.
- **Save Annotated Image**: Export a high-resolution PNG of the image with overlayed annotations.
- **Image Rotation**: Full 90° rotation support with synchronized coordinate mapping.
- **Multi-Object Management**: Sidebar registry to track and manage multiple polygons.
- **Dynamic Themes**: Professional Dark and Light modes with a custom animated sun/moon toggle.
- **Responsive UI**: The UI is made responsive for both desktop and mobile devices.
- **Object Analytics**: Real-time area calculation (PX²) and vertex counting.
- **Micro-Animations**: Success sounds, pulse effects, and animated "marching ants" for active lines.

## 🤖 AI Usage & Decisions
- **SVG vs Canvas**: Chosen for superior DOM event handling on vertices and styling flexibility.
- **AI Partnership**: AI assisted in optimizing complex coordinate rotation math and refining the professional UI aesthetics (Tailwind/Framer Motion).
